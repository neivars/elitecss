Icon

The icon block allows the use of embedded CSS SVG icons lovingly rendered by the community, with ed-icon.

The block itself has no dimensions - it will grow to be contained by whatever dimensions the enclosing parent element affords it. If the dimensions of the enclosing parent aren't square, the icon will offset itself to be centered on whatever the largest axis is.

Using the embedded SVG icons is done through the use of modifiers. Check the Icon Listing for a list of all possible icons. 

<div style="height: 50px; width: 50px">
    <div class="ed-icon ed-icon--icommodity"></div>
</div>
<div style="height: 150px; width: 75px">
    <div class="ed-icon ed-icon--icommodity"></div>
</div>

Disabled Icon

An icon can be shown to have a disabled look by adding the ed-icon--is-disabled modifier to the icon block. 

<div style="height: 50px; width: 50px">
    <div
        class="ed-icon ed-icon--is-disabled ed-icon--icommodity">
    </div>
</div>
