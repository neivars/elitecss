Button

The button component is a rectangular, clickable component that can be composed of quite a few elements.

The simplest button is a block element defined with ed-button, and two elements: ed-button__inner and ed-button__main. The inner element envelops the main corpus of the button, and the main element holds the most prominent content. 

<button class="ed-button">
    <div class="ed-button__inner">
        <div class="ed-button__main">Exit</div>
    </div>
</button>

Button Body

The body of the button can grow more complex, hosting more pieces of secondary information.

Use the ed-button__sub and ed-button__side elements to add subcontent or side content to the button's body. 

<button class="ed-button">
    <div class="ed-button__inner">
        <div class="ed-button__main">Commodities Market</div>
        <div class="ed-button__sub">Agriculture</div>
    </div>
</button>
<button class="ed-button">
    <div class="ed-button__inner">
        <div class="ed-button__main">Refuel</div>
        <div class="ed-button__sub">Main Reserves</div>
        <div class="ed-button__side">75%</div>
    </div>
</button>

Button Head

The most common kind of button in Elite Dangerous has an accompanying icon.

Modify the button block with ed-button--has-head and add the ed-button__head element to give the button a head.

Inside the head element, add your icon block and mix it with the ed-button__icon element so it gets correctly handled by the block. 

<button class="ed-button ed-button--has-head">
    <div class="ed-button__head">
        <div class="ed-icon ed-icon-commodity ed-button__icon"></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Commodities Market</div>
        <div class="ed-button__sub">Agriculture</div>
    </div>
</button>

Disabled Button

A button can be shown as disabled by adding the ed-block--is-disabled modifier to the button block. 

<button class="ed-button ed-button--has-head ed-button--is-disabled">
    <div class="ed-button__head">
        <div class="ed-icon ed-icon-commodity ed-button__icon"></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Remote Workshop</div>
        <div class="ed-button__sub">No Pinned Blueprints</div>
    </div>
</button>

Button with Progress

A button can host a progress block mixed with the ed-button__progress element to make sure it behaves for the button's different states. 

<button class="ed-button ed-button--has-head">
    <div class="ed-button__head">
        <div class="ed-icon ed-icon-commodity ed-button__icon"></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Refuel All</div>
        <div class="ed-button__sub">
            <progress
                class="ed-progress ed-button__progress"
                value="60"
                max="100"
            ></progress>
        </div>
        <div class="ed-button__side">580 CR</div>
    </div>
</button>
<button class="ed-button ed-button--has-head ed-button--is-disabled">
    <div class="ed-button__head">
        <div class="ed-icon ed-icon-commodity ed-button__icon"></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Refuel All</div>
        <div class="ed-button__sub">
            <progress
                class="ed-progress ed-button__progress"
                value="100"
                max="100"
            ></progress>
        </div>
        <div class="ed-button__side">100%</div>
    </div>
</button>

Stacked Button

A button can be stacked with the head on top of its body by using the ed-button--is-stacked modifier with the button block. 

<button class="ed-button ed-button--has-head ed-button--is-stacked">
    <div class="ed-button__head">
        <div
            class="ed-icon ed-button__icon ed-icon--ilivery"
        ></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Livery</div>
    </div>
</button>
<button class="ed-button ed-button--has-head ed-button--is-stacked">
    <div class="ed-button__head">
        <div
            class="ed-icon ed-button__icon ed-icon--iholo-me"
        ></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Holo-Me</div>
        <div class="ed-button__sub">Holo-Me</div>
    </div>
</button>
<button class="ed-button ed-button--has-head ed-button--is-stacked ed-button--is-disabled">
    <div class="ed-button__head">
        <div
            class="ed-icon ed-button__icon ed-icon--iarx-shop"
        ></div>
    </div>
    <div class="ed-button__inner">
        <div class="ed-button__main">Arx Shop</div>
    </div>
</button>
