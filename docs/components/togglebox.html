Togglebox

A togglebox shows a toggleable state of an input: either on or off, true or false.

Stylistically, Elite Dangerous makes no distinction between radio toggles and checkbox toggles, and they both use the same ed-togglebox block.

The block SHOULD be a label tag set with the appropriate id attribute so that the whole clickable area toggles the input tag inside.

Within the block, nest the input tag, with the appropriate attributes set first. If the input tag doesn't come first, the styled checkbox will not respond to toggles.

Use the ed-togglebox__check element to render the custom checkbox and the ed-togglebox__label element to render the accompanying label text. 

<label class="ed-togglebox" for="chemicals">
    <input type="checkbox" name="chemicals" id="chemicals" value="chemicals" />
    <div class="ed-togglebox__check"></div>
    <span class="ed-togglebox__label">Chemicals</span>
</label>
<label class="ed-togglebox" for="legaldrugs">
    <input type="checkbox" name="legaldrugs" id="legaldrugs" value="legaldrugs" />
    <div class="ed-togglebox__check"></div>
    <span class="ed-togglebox__label">Legal Drugs</span>
</label>

Single Select Togglebox (Radio Buttons)

Creating radio buttons is done exactly the same, replacing the input type attribute with "radio", and setting the name attributes of all related radio buttons to the same value so they share state. 

<label class="ed-togglebox" for="galaxy">
    <input type="radio" name="market-scope" id="galaxy" value="galaxy" checked />
    <div class="ed-togglebox__check"></div>
    <span class="ed-togglebox__label">Show Galactic Average</span>
</label>
<label class="ed-togglebox" for="search">
    <input type="radio" name="market-scope" id="search" value="search" />
    <div class="ed-togglebox__check"></div>
    <span class="ed-togglebox__label">Search Markets</span>
</label>
<label class="ed-togglebox" for="suggest">
    <input type="radio" name="market-scope" id="suggest" value="suggest" />
    <div class="ed-togglebox__check"></div>
    <span class="ed-togglebox__label">Suggest Markets</span>
</label>
