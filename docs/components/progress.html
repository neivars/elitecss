Progress

The progress block quantatively shows progression towards a state of completion. A progress bar is defined with the block ed-progress.

Set the usual progress attributes value and max to define the fill of the bar. 

<progress
    class="ed-progress"
    value="75"
    max="100"
></progress>

Disabled Progress

Use the ed-progress--is-disabled modifier on the progress block to show a disabled bar. 

<progress
    class="ed-progress ed-progress--is-disabled"
    value="33"
    max="100"
></progress>
