Section List

A section list block holds a list of section blocks.

Use the ed-section-list block to define a section list, and nest ed-section-list__items within for each section.

Main Section List

A section list modified to have added flair to make it pop out as a main section of the UI.

Use the ed-section-list--main modifier along with the section list block to make it pop out as a main section list.
