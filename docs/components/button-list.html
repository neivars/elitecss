Button List

A section list block holds a list of buttons stacked vertically.

Use the ed-button-list block to define a button list. Its children will be the items within the list. 

<div class="ed-button-list">
    <button class="ed-button ed-button--headed">
        <div class="ed-button__head">
            <div class="ed-button__icon ed-icon ed-icon--icommodity"></div>
        </div>
        <div class="ed-button__inner">
            <div class="ed-button__main">Commodities Market</div>
            <div class="ed-button__sub">Agriculture</div>
        </div>
    </button>
    <button class="ed-button ed-button--headed">
        <div class="ed-button__head">
            <div class="ed-button__icon ed-icon ed-icon--imissionboard"></div>
        </div>
        <div class="ed-button__inner">
            <div class="ed-button__main">Mission Board</div>
        </div>
    </button>
    <button class="ed-button">
        <div class="ed-button__inner">
            <div class="ed-button__main">Exit</div>
        </div>
    </button>
</div>

Horizontal Button List

Use the ed-button-list--horizontal modifier to stack the button list horizontally. This is useful for the stacked version of the buttons. 

<div class="ed-button-list ed-button-list--horizontal">
    <button class="ed-button ed-button--headed ed-button--stacked">
        <div class="ed-button__head">
            <div class="ed-button__icon ed-icon ed-icon--ilivery"></div>
        </div>
        <div class="ed-button__inner">
            <div class="ed-button__main">Livery</div>
        </div>
    </button>
    <button class="ed-button ed-button--headed ed-button--stacked">
        <div class="ed-button__head">
            <div class="ed-button__icon ed-icon ed-icon--iholo-me"></div>
        </div>
        <div class="ed-button__inner">
            <div class="ed-button__main">Holo-Me</div>
        </div>
    </button>

        </div>

    </button>
</div>
